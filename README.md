# ECommerceApp
ECommerce App built with SwiftUI

# Features
    Custom Tab bar
    Static Categories Slider
    Recommended & best selling products  based on ApiResponse
    
    
# Project 
    Project Architecture : MVVM
    API USED : https://fakestoreapi.com/
    Library used : KingFisher
